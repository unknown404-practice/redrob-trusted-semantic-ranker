# RedRob Trusted Semantic Ranker

Welcome to the **RedRob Trusted Semantic Ranker** repository, developed for the India Runs Data and AI Challenge.

## Overview
This repository contains the core Python engine (`rank.py`) designed to efficiently process, score, and rank a massive dataset of 1 Lakh (100,000) candidates using semantic search and advanced ranking logic. The engine is highly optimized, capable of accurately scoring the candidates in approximately 27 seconds on a standard CPU.

## Repository Contents
- **`rank.py`**: The main Python engine that powers the semantic ranking and scoring logic.
- **`top_10_veterans.csv`**: The targeted output showing the top 10 veteran candidates ranked by the engine.
- **`team_gemini.xlsx` & `team_gemini.csv`**: The final output files containing the team's specific submission data and processed results.

*(Note: The raw 1 Lakh candidate dataset and heavy pre-computed embeddings are kept local to optimize repository performance and are not uploaded here.)*
